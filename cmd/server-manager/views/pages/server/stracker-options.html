sTracker Options

sTracker is the popular server side app companion to pTracker, for lap and race tracking.

sTracker is not developed or maintained by {{ if $.IsHosted }}Emperor Servers{{ else }}the Server Manager authors{{ end }}. If you have questions related to the operation of the sTracker tool itself, please ask on the sTracker support thread.

  3. If you are on Linux, extract the stracker_linux_x86.tgz folder. (e.g. tar xfvz stracker_linux_x86.tgz)
  4. Make sure that you have set UDP Plugin Address and UDP Plugin Local Port in Server Options
